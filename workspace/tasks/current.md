## Rules

- Update this file when the founder adds, completes, defers, blocks, or cancels a task.
- If a task is moved from `## Backlog` or `## Backlog with due date` into `## Today`, remove the backlog copy in the same edit so it only exists in one active place.
- Heartbeats should read from this file instead of repeating stale task text.
- Add each founder-owned meeting / call as a task for the day when useful.
- When preparing today's tasks, use `## Every weekday` as the baseline unless the founder says otherwise.
- Use `YYYY-MM-DD` for all-day due dates and `YYYY-MM-DD HH:MM TZ` for timed due dates.
- The daily task-prep flow should also scan `## Recurring reminders` and add any reminder that is due that day into `## Today` without deleting the recurring source item.
- Overlap between `## Today`, `## Every weekday`, and `## Recurring reminders` is expected when today's list has already been explicitly prepared.
- Use `{{ASSISTANT_NAME}}:` as the formal ownership marker for tasks assigned to the assistant.
